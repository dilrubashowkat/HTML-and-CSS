PROJECT WORKS

• Determining barriers in learning MATLAB programming. (Winter 2017)
Role: It was a team project and as a leader I made major study design decisions. I also performed statistical data analysis.
Process: It was an observational lab study with 6 users having no previous experience in MATLAB. Users were asked to think aloud during the task session. We screen record the entire session.
Results: All users reported to have difficulty understanding the code (syntax and complexity) provided in the tools help option. This finding can be used to improve usability of MATLAB tools help feature.
